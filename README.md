## Getting started

Build the PHP docker image with
```shell
docker compose build
```

Install composer dependencies
```shell
./script/composer install
```

Test everything is working with
```shell
./script/composer csrun
```

## Running the tests
```shell
./script/composer test
```

## Code quality
```shell
./script/composer csfix
```
